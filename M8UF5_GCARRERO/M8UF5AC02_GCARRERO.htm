HTML

HTML sòn les sigles d'Hipper Text Markup Language, que podria traduir-se com llenguatge de marques o marcat d'hipertext.

El llenguatge HTML s'utilitza per a crear pàgines web. Ès molt mes senzill de veure el codi HTML d'una pàgina web, l'opció exacta canvia d'un navegador a un altre i també pot canviar d'una versió a una altra en un mateix navegador, però solen tenir una opció similar.

HTML està format d'etiquetes que s'escriuen entre els simbols menor què i major què.

Història d'HTML

Els principis del llenguatge HTML venen de l'any 1991, quan Tim Berners-Lee va crear la primera pàgina web.

Versions d'HTML

Tim Berners-Lee va definir la primera vaersió d'HTML a l'any 1991.

A l'actualitat, la darrera versió d'HTML és HTML5.
